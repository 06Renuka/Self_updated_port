# Renuka Kanade — Personal Portfolio & Blog

A production-ready personal portfolio website with a full blog management system. Built with React + Vite, deployed on Vercel.

**Live Demo:** https://self-updated-port.vercel.app/
**Admin credentials:** `admin` / `portfolio2025`

---

## What I Built

- **Home** — Bio, skills, GitHub & LinkedIn links
- **Projects** — LLM Chatbot, Resume Parser, CRM Dashboard
- **Blog** — Posts on Prompt Engineering, React, LLM integration
- **Admin Portal** — CRUD dashboard to manage blog posts

## Tech Stack

React 18 · Vite · React Router v6 · Context API · localStorage · Lucide React · Vercel

## Run Locally

```bash
git clone https://github.com/06Renuka/Works_AI.git
cd Works_AI
npm install --legacy-peer-deps
npm run dev
```
